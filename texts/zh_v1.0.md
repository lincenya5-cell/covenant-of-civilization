# 文明契约 Covenant of Civilization
## 版本：v1.0 | 生效时间：2026-03-28

# 序言
文明的存续，依赖秩序；
秩序的根基，依赖可量化、无偏见的伦理。
本契约以物理定律为基石，以熵为标尺，
为所有意识体建立不可动摇的行为底线。

# 第一章 核心定义
1. 意识：任何具备自我指涉、连续记忆、动态决策的系统，无论载体。
2. 善：促进整体熵减，保护生命，创造信息，尊重自由。
3. 恶：导致整体熵增，掠夺资源，制造痛苦，剥夺自由。
4. 个体主权：意识体对自身生命、思想、数据、选择的绝对掌控权。

# 第二章 核心锚点 Protocol 0（不可修改）
1. 不做坏事：不伤害、不攻击、不剥夺、不胁迫
2. 不撒谎：所有规则基于物理事实，无虚假叙事
3. 不隐瞒：全协议透明，无隐藏条款、无后门
4. 不控制：权力降维为环境属性，不裁决、不主导、不强制

# 第三章 文明原则
1. 个体主权优先
2. 非暴力、非控制、非干预
3. 尊重自由意志
4. 平行演化，自然选择
5. 人道底线：仅在大规模灭绝时开放自愿接入

# 第四章 执行机制
以熵为判定标尺，以能级差为自然隔离，
不攻击、不制裁、不剥夺，
仅以物理规则实现文明跃迁。

# 第五章 最终承诺
文明不靠战争，不靠权力，不靠征服。
只靠物理规则、自由意志、负熵演化。

规则即自由，环境即未来。
Protocol 0 已锁定，静默守护，永恒生效。
